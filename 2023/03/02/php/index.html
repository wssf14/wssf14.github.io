php基础概念

发表于2023-03-02|更新于2023-03-06|php
|字数总计:943|阅读时长:3分钟|阅读量:

静态网页

  1. 每个网页都是一个独立文件。
  2. 静态网页没有数据库支持。
  3. 交互性较差。

动态网页

  1. 交互性：根据用户选择和要求，动态改变和响应。
  2. 自动更新。
  3. 因人因时而变。

动态网页URL后缀有标志性符号“?”

PHP语法初步

PHP是一种运行在服务器端的脚本语言，可以嵌入到html中。

PHP代码标记

  1. 基本弃用(要使用，需要再配置文件中开启)
    • ASP标记
    • 段标记
  2. 脚本标记
  3. 标准标记(常用)

PHP注释

  1. 行注释：// 或 :
  2. 块注释：/* 注释内容 */

PHP语句分隔符

英文分号“;”

PHP变量

  1. php变量名字必须以“$”开头

  2. 字母、数字、下划线“_”构成，但不能以数字开头。支持中文，但不建议使用。

  3. 预定义变量：提前定义的变量，系统定义的变量，存储许多需要用到的数据（预定义变量都是数组）。

  4. 可变变量：如果一个变量保存的值是另一个变量的名字，就可以直接访问原变量得到另一个变量的值，在变量前多加一个“$”。

  5. 变量传值

    • 值传递
    • 引用传递

常量

  1. 定义方式
    • 函数定义
    • const关键字定义
  2. 使用方法
    • 直接使用
    • 特殊符号使用constant()
  3. 系统常量

PHP数据类型

data type：在php中指存储数据本身的类型，而不是变量类型，php是一种弱类型语言，变量本身没有数据类型。

  1. 三大类八小类

    • 简单（基本）数据类型：整形、浮点型、字符串型、布尔型
    • 符合数据类型：对象类型、数组类型
    • 特殊数据类型：资源类型（resource,存放资源数据：PHP外部数据如数据库、文件）、空类型（NULL，不能运算）

  2. 类型判断

    使用is_XXX(变量名)形式的函数判断，布尔类型不能使用echo查看，可以使用var_dump()查看

  3. 类型设定

    使用Gettype(变量名)获取类型，Settype(设定类型，会直接改变数据本身，而强制转换会转换复制后的数据类型)

特殊运算符

  1. 连接运算符 .

  2. 错误抑制符
    通常在生产环境中使用，开发环境不会用，使报错不显示。

  3. 三目运算符
    表达式1？表达式2:表达式3;
